PROJELER
Chat Clone App

- Whatsapp benzeri bir chat uygulaması geliştirildi. Telefon numarası ile uygulamaya kayıt olunmaktadır. Uygulama ile telefon rehberinde olan ve uygulamaya kayıtlı olan kişiler bu uygulama üzerinde listelenmektedir. Ve bu uygulamadaki kullanıcılara yazı, resim ve ses mesajları atılabilmektedir. Ayrıca durum güncellemesi de yapılabilmektedir. Uygulama üzerinde atılan durumu rehberdeki kullanıcılar görebilmekte ve bu durum 24 saat süre ile veritabanı üzerinde saklanmaktadır. Aynı zamanda kullanıcılar ile mesajlaşırken mesaj geçmişi kullanıcı profilinde gösterilmekte ve bu profil üzerinde kullanıcıyı engelleme ve şikayet etme gibi özellikler bulunmaktadır. Uygulama üzerinde mesajlama arkaplanını varsayılan olarak veya kişiye özel bir şekilde düzenlenyerek görsel düzenlemeler yapılabilir.

- Uygulama yapılırken programlama dili olarak JAVA kullanıldı. Kullanıcı verileri Firebase üzerinde tutuldu. Uygulamaya kayıt olma giriş yapma işlemleri Firebase Authentication ile yapıldı.



Stok Takip App

- Stok takibi yapılabilen bir uygulamadır. Uygulama üzerinde bir profil oluşturup kullanılmaya başlanabilir. Uygulama ile müşteri, tedarikçi, müşteriye satılan ürünler, tedarikçiden satın alınan ürünler, ek giderler ve kasaya eklenen veya çıkan ücretler saklanabilir. Uygulama üzerinde tedarikçi ve müşteriler aranıp bu müşterilere sms mesajı atılabilir.

- Uygulama yapılırken programlama dili olarak JAVA kullanıldı. Kullanıcı verileri Firebase üzerinde tutuldu. Uygulamaya kayıt olma giriş yapma işlemleri Firebase Authentication ile yapıldı.



NASA API Mars App

- NASA API üzerinden rover araçların çektiği resimleri listeleyen bir uygulamadır. Bu uygulamada 3 çeşit rover bulunmaktadır ve her bir rover aracın sahip olduğu kameralara göre filtrelenip resimlere ulaşılabilir.

- Uygulama yapılırken programlama dili olarak KOTLIN kullanıldı. Veriler NASA API üzerinden çekildi ve resimler pagination ile recyclerview üzerinde listelendi. Rest Api client olarak ise Retrofit/Okhttp kullanıldı.



Sosyal Medya Web Site

- Web sitesi üzerinde durum ve resim paylaşılabilir. Arkadaş eklenip isteği kabul edilen arkadaşlara mesaj gönderilebilir. Bu arkadaşların paylaştığı durumlara ve resimlere beğeni atılabilir. Aynı zamanda profil kısmında kullanıcı kendi hakkında bilgiler verip bu bilgiler hakkında arkadaş önerilerini görebilir.

- Web sitesi yapılırken JAVA, JSF, HTML, CSS kullanıldı. Kullanıcı verileri local veritabanında tutuldu ve MYSQL kullanıldı. Uygulama 2 ekip arkadaşı ile yapıldı.
